qwak

qwak is an APACHE licensed library written in Python designed to provide a simple to use API wrapper for MPOS.

Installation:

From source use

$ python setup.py install

or install from PyPi

$ pip install qwak

Configuration:

Create a ~/.qwak_config.py file and fill in your pool URL and API key like so:

#!/usr/bin/env python
# -*- coding: utf-8 -*-

config = {
    'pool_url': 'http://example.com',
    'api_key' : '12345abcde'
}
